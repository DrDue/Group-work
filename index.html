Git

Lokal start

Projekt start

Åben cmd
  • mkdir projektnavn
  • cd projektnavn
  • git init

Tilføj filer

  • git add .
    • eller
  • git add filnavn.js

Commit

  • git commit -m "besked"

Del lokalt arbejde

  • cd projektnavn
  • git remote add origin https://youruserid@bitbucket.org/youruserid/projectname.git
  • git push -u origin master
